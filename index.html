HTML/CSS Cheat Sheet

HTML

HTML, short for HyperText Markup Language, is the foundation of all web pages and was created to define the structure of a web page.

HTML Structure

HTML is organised into a family tree structure. HTML elements can have parents, grandparents, siblings, children, grandchildren etc.

HTML Tags

In HTML tags are commonly defined as a set of characters constituting a formatted command for a web page.

Commonly Used Elements

Element

Name

Description
<a> Anchor Element The anchor element is used to create hyperlinks in an HTML document.
<audio> Audio Used for adding audio content to a page. HTML supports the following audio formats; MP3, WAV and OGG.
<b> Bold This element is used to draw the readers attention to the marked text by making it bold.
<body> Body This represents the content of an HTML document. There can only be one body element in a web page.
<br> Line Break Creates a line break in text and is used when a division of text is required and does not require a closing tag.
<button> Button The button element represents a button meant to be clicked by the user.
<div> Div The div (division) element is used as a container that divides an HTML document into sections.
<em> Emphasise This represents text that is emphasised, by default browsers will show the enclosed in italics.
<form> Form The form element represents an interface to collect and submit user supplied information.
<h1>-<h6> Heading Elements HTML can use six different levels of headings, highest being <h1>.
<head> Head This element represents a collection of metadata related to the current document.It's an immediate child of the HTML element.
<HTML> HyperText Markup Language Used to give content to a web page and instructs browsers on how to display content.
<img> Image Used to display an image on a web page. The image element has no closing tag, meaning it cannot have elements inside of it.
<input> Input This is used to create an interactive element, usually within a form. The input element has no closing tag, meaning it cannot have elements inside of it.
<li> List Represents a single list item, this element must be wrapped in an <ol>, <ul> or <menu> element.
<link> Link This element connects the current web page with an external file such as a stylesheet or a library.
<menu> Menu Represents an unordered list of elements with more semantic meaning than a regular <ul> element.
<ol> Ordered List Represents a ordered list of items.
<option> Option The option element represents one possible selection in the dropdown created by the <select> tag.
<output> Output This element displays the result of calculation or user action. Commonly used inside a <form> to show the result of user input.
<p> Paragraph The paragraph element contains and displays blocks of text.
<script> Script This element is used to insert runnable code into a document, usually JavaScript. It can be used in HTML or to load on an external script from another service.
<select> Select This element is nomally used in a form to create a dropdown list where the user can select from a number of <option> elements.
<source> Source This element allows for multiple sources to be associated with media tags. This is a child of <audio>, <video> or <picture> tags and has at least one attribute.
<span> Span This element is used for grouping text or elements together for styling and scripting.This is an inline element usually wrapped inside another element.
<strong> Strong This element is used to identify text that is very important or urgent.
<style> Style The style element applies CSS rules to an HTML document and must be included inside the <head> element of the document.
<table> Table This represents an interface for adding tabular data to a page.
<tr> Table Row This is used to display data within a row in a horizontal table.
<th> Table Header This element is used to display headings for specific columns within a table.
<td> Table Data This is used to display individual data within a specific row/ column position in a table.
<ul> Unordered List This element represents an unordered list of items.
<video> Video This represents an interface for adding video content to a page.
